## Description

Extend podkit to be able to manage and sync with additional device types. This includes the transfer mode system with device-aware artwork handling for non-database-artwork devices. Includes embedded artwork resizing, sidecar artwork creation, and the Echo Mini device adapter. This is the first time this project will be supporting more than libgpod compatible devices. Builds on the DeviceCapabilities interface from the iPod milestone. See DOC-012 (future behavior matrices) and DOC-013 (Echo Mini capabilities).
